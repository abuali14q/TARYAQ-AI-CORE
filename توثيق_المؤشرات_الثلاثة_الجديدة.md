## 1️⃣ مؤشر التضخم المتوقع (Inflation Rate)

### 📌 الوصف
معدل التضخم المتوقع خلال فترة المشروع، محسوب بناءً على:
- معدل التضخم الأساسي للسعودية (3.5% سنوياً)
- مدة المشروع
- الموقع الجغرافي

### 🔢 الصيغة الحسابية

#### الخطوة 1: حساب معدل التضخم الأساسي
```
معدل_التضخم = 3.5% (معدل افتراضي للسعودية)
```

#### الخطوة 2: تعديل حسب مدة المشروع
```
إذا كانت المدة > 365 يوم:
   معدل_التضخم += (المدة/365 - 1) × 0.8%
وإلا:
   معدل_التضخم = 3.5% × (المدة/365)
```

#### الخطوة 3: تعديل حسب المنطقة
```
إذا كانت المنطقة حارة (الرياض، نيوم):
   معدل_التضخم × 1.15
وإذا كانت متوسطة (جدة):
   معدل_التضخم × 1.10
وإلا:
   معدل_التضخم بدون تعديل
```

#### الخطوة 4: تحديد الحدود
```
معدل_التضخم_النهائي = MAX(0.5%, MIN(معدل_التضخم، 8%))
```

### 📊 الأمثلة العملية

#### مثال 1: مشروع قصير في الرياض
```
المدة: 60 يوم
المنطقة: الرياض (حارة)

الحساب:
1. معدل_أساسي = 3.5%
2. تعديل_المدة = 3.5% × (60/365) = 0.575%
3. تعديل_المنطقة = 0.575% × 1.15 = 0.66%
4. النتيجة = 0.66% ≈ 0.66%
```

#### مثال 2: مشروع طويل في الشرقية
```
المدة: 730 يوم (سنتان)
المنطقة: الشرقية (عادية)

الحساب:
1. معدل_أساسي = 3.5%
2. تعديل_المدة = 3.5% + (2 - 1) × 0.8% = 4.3%
3. تعديل_المنطقة = 4.3% × 1.0 = 4.3%
4. النتيجة = 4.3% ≈ 4.3%
```

### 🎨 ترميز اللون
- 🔵 **أزرق:** معدل تضخم منخفض (< 5%)
- 🟡 **أصفر:** معدل تضخم متوسط (5% - 7%)
- 🔴 **أحمر:** معدل تضخم عالي (≥ 7%)

### 💡 التفسير والتوصيات
- **معدل منخفض (< 5%):** يشير إلى استقرار اقتصادي وتضخم معتدل
- **معدل متوسط (5% - 7%):** يتطلب مراجعة الميزانية والعقود
- **معدل عالي (≥ 7%):** ضرورة تضمين بند تضخم في العقد مع المقاول

## 2️⃣ مؤشر البصمة الكربونية (Carbon Footprint)

### 📌 الوصف
إجمالي انبعاثات غازات الدفيئة (CO₂) المتوقعة من المشروع، محسوب بناءً على:
- حجم المشروع
- الموقع الجغرافي (درجات الحرارة)
- مدة المشروع
- كفاءة العمالة

### 🔢 الصيغة الحسابية

#### الخطوة 1: الانبعاثات الأساسية حسب حجم المشروع
```
حجم المشروع                 الانبعاثات الأساسية
─────────────────────────────────────────
صغير (Small)               50 طن CO₂
متوسط (Medium)             150 طن CO₂
كبير (Large)               400 طن CO₂
ضخم (Mega)                 800 طن CO₂
```

#### الخطوة 2: تعديل حسب المنطقة الجغرافية
```
المنطقة                     معامل التعديل    السبب
─────────────────────────────────────────────────────
الرياض / نيوم (حارة)        × 1.30          تكييف مكثف
جدة (متوسطة)                × 1.10          تكييف معتدل
الشرقية / عسير (بردة)        × 1.00          بدون تعديل
```

#### الخطوة 3: تعديل حسب مدة المشروع
```
البصمة = البصمة × (المدة / 180)
```
*حيث 180 يوم تعتبر المدة المرجعية*

#### الخطوة 4: تعديل حسب كفاءة العمالة
```
البصمة = البصمة × (1.1 - كفاءة_العمالة × 0.1)
```

#### الخطوة 5: الصيغة الكاملة
```
البصمة_النهائية = 
  البصمة_الأساسية 
  × معامل_المنطقة 
  × (المدة / 180) 
  × (1.1 - كفاءة_العمالة × 0.1)
```

### 📊 الأمثلة العملية

#### مثال 1: مشروع صغير في الشرقية
```
الحجم: صغير
المنطقة: الشرقية
المدة: 90 يوم
كفاءة العمالة: 80%

الحساب:
1. البصمة_أساسية = 50 طن CO₂
2. معامل_منطقة = 1.0
3. معامل_مدة = 90 / 180 = 0.5
4. معامل_عمالة = 1.1 - 0.8 × 0.1 = 1.02
5. النتيجة = 50 × 1.0 × 0.5 × 1.02 = 25.5 طن CO₂
```

#### مثال 2: مشروع ضخم في الرياض
```
الحجم: ضخم
المنطقة: الرياض
المدة: 365 يوم
كفاءة العمالة: 85%

الحساب:
1. البصمة_أساسية = 800 طن CO₂
2. معامل_منطقة = 1.3
3. معامل_مدة = 365 / 180 = 2.03
4. معامل_عمالة = 1.1 - 0.85 × 0.1 = 1.015
5. النتيجة = 800 × 1.3 × 2.03 × 1.015 = 2,129 طن CO₂
```

### 🎨 ترميز اللون
- 🟢 **أخضر:** بصمة منخفضة (< 150 طن CO₂) - استدامة عالية
- 🟡 **أصفر:** بصمة متوسطة (150 - 400 طن CO₂) - استدامة معتدلة
- 🔴 **أحمر:** بصمة عالية (> 400 طن CO₂) - استدامة منخفضة

### 💡 معدلات لقياس الأداء
- **هدف السعودية 2030:** تجنب 130 مليون طن CO₂ سنوياً
- **مختبر LEED:** انبعاثات منخفضة = < 50 طن CO₂
- **التصنيف الدولي:** مشروع أخضر = < 100 طن CO₂
- **التصنيف البرونزي:** 100 - 300 طن CO₂
- **التصنيف الفضي:** 300 - 600 طن CO₂

### ♻️ التوصيات للتقليل
1. **استخدام المواد المستدامة:** تقليل ~ 15%
2. **تحسين كفاءة العمالة:** كل 10% زيادة = تقليل 1%
3. **تقصير مدة المشروع:** تقليل ساعات العمل والنقل
4. **استخدام الطاقة المتجددة:** تقليل ~ 25%
5. **إعادة تدوير المواد:** تقليل ~ 10%

## 3️⃣ مؤشر مطابقة الاشتراطات (Requirements Compliance)

### 📌 الوصف
نسبة مئوية لقياس مدى استيفاء المشروع للاشتراطات والمعايير المطلوبة، محسوب من 5 معايير أساسية:
1. كفاءة العمالة (2 نقطة)
2. ملائمة الميزانية (2 نقطة)
3. واقعية المدة الزمنية (2 نقطة)
4. تقييم المخاطر (2 نقطة)
5. الامتثال البيئي (2 نقطة)

### 🔢 نظام التقييم

#### معيار 1: كفاءة العمالة (Labor Efficiency)
```
كفاءة > 85%   → 2 نقطة (ممتاز)
كفاءة > 70%   → 1 نقطة (جيد)
كفاءة ≤ 70%   → 0 نقطة (ضعيف)
```

#### معيار 2: ملائمة الميزانية (Budget Adequacy)
```
الميزانية/اليوم > 50,000 SAR   → 2 نقطة (ممتاز)
الميزانية/اليوم > 20,000 SAR   → 1 نقطة (جيد)
الميزانية/اليوم ≤ 20,000 SAR   → 0 نقطة (ضعيف)
```

#### معيار 3: واقعية المدة الزمنية (Duration Realism)
```
المدة ≤ 365 يوم (سنة)   → 2 نقطة (واقعي جداً)
المدة ≤ 730 يوم (سنتان) → 1 نقطة (منطقي)
المدة > 730 يوم        → 0 نقطة (غير واقعي)
```

#### معيار 4: تقييم المخاطر (Risk Assessment)
```
معدل الخطر < 30%   → 2 نقطة (آمن جداً)
معدل الخطر < 60%   → 1 نقطة (متقبل)
معدل الخطر ≥ 60%   → 0 نقطة (عالي الخطر)
```

#### معيار 5: الامتثال البيئي (Environmental Compliance)
```
البصمة < 100 طن CO₂    → 2 نقطة (أخضر جداً)
البصمة < 300 طن CO₂    → 1 نقطة (متوازن)
البصمة ≥ 300 طن CO₂    → 0 نقطة (ضعيف)
```

### 📊 حساب النسبة المئوية

```
النقاط_الكلية = معيار1 + معيار2 + معيار3 + معيار4 + معيار5
النقاط_القصوى = 10

نسبة_الامتثال = (النقاط_الكلية / 10) × 100%

المعيار:
نسبة ≥ 60%   → ✅ مطابق (Compliant)
نسبة < 60%   → ❌ غير مطابق (Non-Compliant)
```

### 📊 الأمثلة العملية

#### مثال 1: مشروع ممتاز (تطابق 90%)
```
المدخلات:
- كفاءة عمالة: 92%
- ميزانية: 1,500,000 SAR / 30 يوم = 50,000/يوم
- المدة: 30 يوم
- معدل الخطر: 25%
- البصمة: 75 طن CO₂

النتيجة:
1. كفاءة عمالة: 92% > 85% → 2/2 ✓
2. ميزانية كافية: 50,000/يوم = حد الممتاز → 2/2 ✓
3. المدة واقعية: 30 يوم < 365 → 2/2 ✓
4. المخاطر منخفضة: 25% < 30% → 2/2 ✓
5. امتثال بيئي: 75 < 100 → 2/2 ✓

الإجمالي = 10/10 = 100% → ✅ مطابق تماماً
```

#### مثال 2: مشروع متوسط (تطابق 50%)
```
المدخلات:
- كفاءة عمالة: 75%
- ميزانية: 600,000 SAR / 60 يوم = 10,000/يوم
- المدة: 60 يوم
- معدل الخطر: 50%
- البصمة: 250 طن CO₂

النتيجة:
1. كفاءة عمالة: 75% بين 70%-85% → 1/2 ⚠️
2. ميزانية: 10,000/يوم < 20,000 → 0/2 ✗
3. المدة واقعية: 60 يوم < 365 → 2/2 ✓
4. المخاطر: 50% بين 30%-60% → 1/2 ⚠️
5. امتثال بيئي: 250 بين 100-300 → 1/2 ⚠️

الإجمالي = 5/10 = 50% → ❌ غير مطابق
```

### 🎯 مستويات التقييم

| النسبة | التصنيف | الحالة | الإجراء المطلوب |
|--------|---------|--------|-----------------|
| 90% - 100% | ممتاز | ✅ مطابق | الموافقة الفورية |
| 70% - 89% | جيد | ✅ مطابق | موافقة مشروطة |
| 60% - 69% | مقبول | ✅ مطابق | توصيات تحسين |
| 40% - 59% | ضعيف | ❌ غير مطابق | تعديلات مطلوبة |
| < 40% | سيء | ❌ غير مطابق | رفض وإعادة صياغة |

### 💡 التوصيات للتحسين

1. **إذا كانت كفاءة العمالة منخفضة:**
   - زيادة التدريب المهني
   - تحسين الإشراف والمتابعة
   - زيادة المكافآت والحوافز

2. **إذا كانت الميزانية غير كافية:**
   - إعادة تقدير التكاليف
   - تقسيم المشروع لمراحل
   - البحث عن تمويل إضافي

3. **إذا كانت المدة غير واقعية:**
   - تقليل نطاق المشروع
   - زيادة عدد الفرق العاملة
   - استخدام تقنيات أسرع

4. **إذا كانت المخاطر عالية:**
   - خطة إدارة مخاطر شاملة
   - آليات حماية قانونية
   - بند طوارئ مالي

5. **إذا كانت البصمة الكربونية عالية:**
   - استخدام مواد صديقة للبيئة
   - تحسين كفاءة الطاقة
   - تقليل مسافات النقل
