## 📊 نماذج الحساب

### 1. حساب التأخير المتقدم
```
التأخير = تأخير_العمالة + تأخير_الطقس + تأخير_الحجم + تأخير_الميزانية + تأخير_المخاطر

حيث:
- تأخير_العمالة = (1 - كفاءة)^1.5 × أيام × 0.35
- تأخير_الطقس = أيام × معامل_المنطقة_الطقسي
- تأخير_الحجم = أيام × 0.05 × معامل_التعقيد
- تأخير_الميزانية = أيام × معامل_ضغط_الميزانية
- تأخير_المخاطر = أيام × نقاط_تعقيد_المخاطر
```

### 2. حساب معدل الخطر
```
معدل_الخطر = (عامل_التأخير×0.45 + عامل_العمالة×0.35 + عامل_الميزانية×0.20) × 1.1

النطاق: 0-95%
```

### 3. حساب معدل التضخم
```
التضخم = معدل_أساسي × معامل_المنطقة × معامل_المدة × معامل_الضغط

المعدل الأساسي: 3.5% سنوياً
```

### 4. حساب البصمة الكربونية
```
الانبعاثات = الأساس × الإقليمي × المدة × معامل_الكفاءة

الأساس: يعتمد على حجم المشروع
```
