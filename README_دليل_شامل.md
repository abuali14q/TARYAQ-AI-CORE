### 3️⃣ **الصيغ_الرياضية_والحسابات.md** 🧮 التفصيلي
**الحجم:** 15-20 صفحة | **الوقت:** 25-40 دقيقة

**المحتوى الرئيسي:**

#### 🔹 الصيغ الأساسية:
1. **حساب التأخير الأساسي**
   ```
   base_delay = (1.0 - sel_labor) × (sel_days × 0.4)
   ```
   - شرح دقيق لكل متغير
   - أمثلة حسابية خطوة بخطوة

2. **حساب تأخير الطقس**
   ```
   weather_delay = sel_days × 0.15 (إذا كانت المنطقة حارة)
   ```
   - الشروط والحالات الخاصة

3. **إجمالي التأخير**
   ```
   P_var = round(base_delay + weather_delay, 1)
   ```

4. **معدل الخطر**
   ```
   risk_percentage = min(int((P_var ÷ sel_days) × 100) + 15, 95)
   ```

5. **تجاوز التكلفة**
   ```
   cost_overrun_val = min(int((P_var ÷ sel_days) × 35), 100)
   ```

#### 🔹 سيناريوهات حسابية:
- **السيناريو الأول:** مشروع متدهور (Project in Crisis)
- **السيناريو الثاني:** مشروع محسّن (Optimized Project)
- أمثلة مع أرقام حقيقية وتفاصيل كاملة

#### 🔹 جدول اختياري:
- مصفوفة الحساب السريع
- نتائج جاهزة لمعاملات مختلفة

**من يستخدمه:**
- 🧮 المتخصصون بالحسابات
- 📐 المهندسون الإنشائيون
- 💻 محللو البيانات
- 🔬 الباحثون الأكاديميون

### 4️⃣ **دليل_الإجراءات_والتوصيات.md** 🚀 العملي
**الحجم:** 25-30 صفحة | **الوقت:** 35-50 دقيقة

**المحتوى الرئيسي:**

#### 🔹 الإجراءات حسب مستوى الخطر:

**مستوى 1: خطر منخفض (0-20%)**
- ✅ المراقبة الروتينية
- ✅ الحفاظ على الأداء
- ✅ التخطيط للمرحلة القادمة

**مستوى 2: خطر منخفض-متوسط (21-40%)**
- 🟡 تحسينات العمالة
- 🟡 تعديلات الطقس
- 🟡 ضمان توفر المواد
- 🟡 خطة عمل أسبوعية

**مستوى 3: خطر متوسط (41-60%)**
- 🟠 تعزيز القوة العاملة بـ 20-30%
- 🟠 برنامج تحريز مكثف
- 🟠 إضافة معدات إضافية
- 🟠 جدول عمل معجل
- 🟠 تقارير أسبوعية

**مستوى 4: خطر عالي جداً (61-95%)**
- 🔴 اجتماع طاريء للإدارة العليا
- 🔴 زيادة العمالة بنسبة 50-100%
- 🔴 عمل 24 ساعة (نوبات)
- 🔴 خطة إنقاذ فورية
- 🔴 متابعة يومية

#### 🔹 الإجراءات حسب نوع المشكلة:
1. **ضعف كفاءة العمالة** 👷
   - التشخيص والعلامات التحذيرية
   - طرق القياس والحلول
   - جدول زمني للتحسين

2. **تأثير الطقس القاسي** 🌡️
   - الحلول الفورية
   - تعديلات الجدول
   - معدات الحماية

3. **نقص المواد والموارد** 📦
   - نظام إدارة المواد
   - موردون بديلون
   - استراتيجيات التخزين

4. **مشاكل سلسلة التوريد** 🔗
   - تحسينات فورية
   - تحسينات قصيرة المدى
   - استراتيجيات طويلة المدى

#### 🔹 جداول وخطط جاهزة:
- جدول المتابعة الأسبوعي
- خطط عمل معجلة
- قوائم تحقق شاملة
- نماذج الاجتماعات

**من يستخدمه:**
- 👔 مديرو المشاريع التنفيذيون
- 🎯 قادة الفريق الميدانيون
- 📋 منسقو المشاريع
- ⚙️ المسؤولون عن تنفيذ الخطط
